Testing BaseGeometry class:

1. Test area method (should raise Exception):
--------------------------------------------------
from geometry import BaseGeometry

bg = BaseGeometry()
try:
    bg.area()
except Exception as e:
    print("[area] Exception:", e)

2. Test integer_validator with valid input:
--------------------------------------------------
try:
    bg.integer_validator("width", 10)
    print("[integer_validator] Passed valid input.")
except Exception as e:
    print("[integer_validator] Exception:", e)

3. Test integer_validator with non-integer input:
--------------------------------------------------
try:
    bg.integer_validator("height", "10")
except Exception as e:
    print("[integer_validator] Exception:", e)

4. Test integer_validator with zero or negative:
--------------------------------------------------
try:
    bg.integer_validator("size", 0)
except Exception as e:
    print("[integer_validator] Exception:", e)

try:
    bg.integer_validator("size", -5)
except Exception as e:
    print("[integer_validator] Exception:", e)
